AI Tools for visual creation and critique.

A CLI toolbelt that generates cat imagery, captures live website screenshots, and routes them to Gemini for grounded design feedback.

Explore toolchain View workflow

What we built

Command line modules with a cosmic edge.

Each command is designed to ship quickly while preserving visual rigor.

images

Cat image generator

Creates high-resolution cat visuals using OpenAI, saving each PNG into the /images directory with timestamped naming.

  • Batch-friendly generation
  • Prompt guardrails for cats
  • Local file output
design-review

Screenshot critique pipeline

Captures a site with Playwright, sends the screenshot to Gemini, and writes structured feedback into /reviews.

  • Viewport + full-page capture
  • Custom rubric prompts
  • Markdown feedback logs
research

Web search intelligence

Runs a web search with OpenAI and stores a summarized report in the /library index.

  • Source-aware summaries
  • Offline and allow-list modes
  • Structured archive

Quick command

Run the review pipeline.

npm run dev -- design-review https://example.com \
  --viewport 1440x900 \
  --full-page \
  --prompt "Audit layout, typography, contrast, and motion."

Feedback and screenshots are stored automatically.
